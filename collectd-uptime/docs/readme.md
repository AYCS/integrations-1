### Uptime Metrics

Use the [uptime plugin](https://collectd.org/wiki/index.php/Plugin:Uptime) to collect metrics about system uptime.  This plugin shows seconds elapsed since the system was last started.

### Version information

| Software  | Version        |
|-----------|----------------|
| collectd  |  4.7 or later  |
